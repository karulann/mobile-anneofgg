# Roadmap

## Product Goal

Build a family friendly Android ebook reader app for the Anne of Green Gables series.

The app should be:

- Easy to use
- Attractive
- Readable for children, adults, and elderly users
- Offline first
- Monetizable without aggressive ads
- Suitable for Google Play publication

---

## Phase 0: Foundation

### Goal

Prepare the project correctly before feature development.

### Tasks

- [ ] Confirm Android Studio project setup
- [ ] Confirm Kotlin and Jetpack Compose are enabled
- [ ] Confirm minimum SDK
- [ ] Correct package name for production
- [ ] Initialize Git
- [ ] Create GitHub repository
- [ ] Push first commit
- [ ] Add project tracking documentation

## Phase 2: Safe Monetization

### Goal

Add income without damaging the reading experience or increasing Google Play policy risk.

### Features

- [ ] AdMob banner on Home screen
- [ ] AdMob banner on Book Detail screen
- [ ] No ads inside Reader screen
- [ ] No interstitial ads in MVP
- [ ] No rewarded ads in MVP
- [ ] One time Remove Ads purchase
- [ ] Data Safety declaration
- [ ] Ads declaration
- [ ] Target Audience declaration
